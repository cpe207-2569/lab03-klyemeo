Skills

Programming

ผมเขียน Python และ C++ ได้นิดหน่อย และเชี่ยวชาญ eV3G เป็น block code สำหรับหุ่นยนต์เลโก้

Image Processing

ผมเคยมีประสบการณ์ใช้ YOLO ตรวจจับพวกผลผลิตทางการเกษตร จำแนกป้ายจราจรเบื้องต้น การปรับภาพให้ได้เฉพาะวัตถุที่ต้องการ

3D design and printing

ผมสามารถขึ้นชิ้นงาน สามมิติผ่านโปรแกรม ออกแบบชิ้นส่วนสำหรับงานเล็กๆได้ และ ปรับจูนเครื่องปริ้น หรือ วัสดุให้เหมาะกับ งานที่เราจะนำไปใช้สำหรับชิ้นงานนั้นๆ

Control System

ในตลอดการแข่งขันหุ่นยนต์ ที่เคยประสบมา ผมได้นำการประยุกต์ใช้ระบบวัดคุมเข้ามาเป็นตัวช่วยในการควบคุมหุ่นยนต์ให้มีเสถียรภาพมากขึ้น เช่น การเดินตามเส้น การเลี้ยวโค้งด้วย [Gyro Sensor] การวิ่งขนานกับกำแพงโดยใช้ [Ultrasonic Sensor] หรือแม้กระทั่งการเคลื่อนที่แบบ synchronize และการควบคุมแขนกลสำหรับหนีบจับวัตถุด้วยความแม่นยำ
